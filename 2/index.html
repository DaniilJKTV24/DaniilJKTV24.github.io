Создание форм в HTML5

↑ ↓

Общее описание

HTML формы являются элементами управления, которые применяются для сбора информации от посетителей веб-сайта.

Веб-формы состоят из набора текстовых полей, кнопок, списков и других элементов управления, которые активизируются щелчком мыши. Технически формы передают данные от пользователя удаленному серверу. Для получения и обработки данных форм используются языки веб-программирования, такие как PHP, Perl.

Создание формы

Форма начинается с парного тега <form></form>. Тег <form> удерживает все элементы управления формы, которые также называются полями. Атрибуты этого элемента содержат информацию, общую для всех полей формы, поэтому в одну форму нужно включать поля, объединенные логически.

<form name="form_1" action="script/request.php" method="post"></form>

  • Атрибут name. Значение этого атрибута означает имя HTML формы.
  • Атрибут action. Значение этого атрибута отвечает за путь к файлу скрипта, который будет обрабатывать форму. То есть мало ввести данные в форму, их же ведь ещё нужно соответствующим образом обработать, и вот как раз путь к этому файлу-обработчику и находится в значении атрибута action.
  • Атрибут method. У него может быть одно из двух значений: post или get. Этот атрибут определяет способ отправки. Первый способ является скрытой отправкой данных, а второй открытой.

Добавление текстового поля

Добавляется текстовое поле с помощью тега <input>, а точнее с помощью атрибута этого тега type со значением text.

<input type="text" name="firstname" value="Введите фамилию">

  • Атрибут type отвечает за тип элемента ввода. В данном случае, мы указали, что это обычное текстовое поле. В следующих элементах мы будем менять значение этого атрибута.
  • Атрибут name отвечает за имя элемента. Здесь мы указали, что имя данного поля firstname.
  • Атрибут value отвечает за значение этого поля по умолчанию.
Фамилия

Добавление текстового поля для ввода пароля

Однострочное текстовое поле для ввода не обычных символов, а ввода пароля, то есть текст должен быть скрыт звездочками.

<input type="password" name="pass" value="Введите пароль">

Теперь значением атрибута type является password. Остальные атрибуты те же, что и для обычного текстового поля.

Добавление выпадающего списка

Состоит из объявления создания выпадающего списка и добавления элементов списка.

<select name="choice">

<option value="1">Вариант 1</option>

<option value="2">Вариант 2</option>

<option value="3">Вариант 3</option>

</select>

Тег <select> означает начало выпадающего списка. Заканчивается список тегом </select>. Элементы списка создаются с помощью тега <option>. Значение атрибута value означает, какое значение будет у переменной choice, то есть либо choice = 1, либо choice = 2, либо choice = 3. Сразу после окончания описания тега <option> ставится то, что увидит пользователь в выпадающем списке.

Добавление текстовой области

Для добавления текстовой области используется тег <textarea>.

<textarea name="message" rows="10" cols="45">Сообщение</textarea>

Создана текстовая область высотой в 10 строк (значение атрибута rows) и шириной в 45 символов (значение атрибута cols). Внутри этого парного тега указывается текст в текстовой области по умолчанию.

Добавление радиокнопки

Радиокнопка (radio-button) - элемент интерфейса, который позволяет пользователю выбрать один пункт из предопределенного набора (группы). Для создания радиокнопки используется тег <input>.

Выберите что-нибудь одно:

<input type="radio" name="choiceradio" value="1">Вариант 1

<input type="radio" name="choiceradio" value="2">Вариант 2

<input type="radio" name="choiceradio" value="3">Вариант 3

  • Атрибут name отвечает не только за задание имени, но и одна группа радиокнопок должна иметь одно и то же значение атрибута name. Если НЕ сделать одинаковыми имена у этих трёх радиокнопок, то они станут независимыми, а, следовательно, можно будет выбрать сразу несколько вариантов.
  • Атрибут value означает значение переменной choiceradio. Сразу после описания тега <input> идёт текст, который увидит пользователь рядом с соответствующей радиокнопкой.
  • Атрибут value отвечает за значение этого поля по умолчанию.

Добавление переключателей

Переключатели (checkbox) создаются с помощью тега <input>.

Вы согласны с нашими правилами:

<input type="checkbox" name="terms" value="yes">

Если флажок будет стоять, то значение переменной terms будет yes, если не будет стоять, то значение этой переменной будет "", то есть пустой строкой.

Добавление поля для выбора файла

При загрузке файла на сервер, часто приходится использовать поле вставки имени файла, которое создается с помощью тега <input>.

Выберите файл для загрузки:

<input type="file" name="path">

Кнопки

  1. Обычная кнопка

Создается кнопка, которая сама по себе не совершает никаких действий и не отправляет форму на сервер. Это многофункциональная кнопка, управляемая JavaScript (обычно через событие click).

<input type="button" name="start" value="Начать">

Значение атрибута type - button, сообщает браузеру, что это кнопка.
Атрибут name - это имя кнопки.
Значение атрибута value отвечает за текст на кнопке.
